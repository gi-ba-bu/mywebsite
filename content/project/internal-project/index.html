Using ggplot() and dplyr

In this section I take a look at data related to the Covid-19 epidemic. The data is imported from the European Centre for Disease Prevention and Control. First, I show a table for the cumulative deaths and cases for each EU country for which data is available. Numbers are usually up to date.

library(tidyverse)         # for graphing and data cleaning
library(kableExtra)        # for better tables

#read the dataset into “R”. The dataset will be called "data".
data <- read.csv("https://opendata.ecdc.europa.eu/covid19/nationalcasedeath_eueea_daily_ei/csv", na.strings = "", fileEncoding = "UTF-8-BOM")

#extract cumulative deaths and cases for each EU country for which data is available
q <- data %>% 
  group_by(countriesAndTerritories) %>% 
  summarize(deaths = sum(deaths),
            cases = sum(cases)) %>% 
  arrange(desc(deaths))

# plot the table
kbl(q) %>%
   kable_styling(bootstrap_options = c("responsive"))
